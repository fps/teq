int teq::teq::process ( jack_nframes_t  nframes)
protected

Here come all the state transitions that depend on the ticks and not individual frames.

Wrap tick around if we meet the pattern boundary

Wrap aound to loop start if we hit the loop endl

And stop the transport and do nothing if we ran out of the end of the song
